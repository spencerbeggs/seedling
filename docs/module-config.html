Module: config

config

This module provides a unified configuration object where application-level settings can be stored. Properties that are safe to expose in compiled JavaScript should be stored or calculated in the public.js. Settings that you would not want exposed in compiled JavaScript should be stored or calculated in private.js. The Gulp tasks included in this package use Envify and Uglifyify transforms to strip the private code from the Browserify bundle by injecting an enviornmental variable BROWSERIFY with the value of true via Envify.
